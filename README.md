## How to install ?
 
* **Clone the repo via git**
    ```bash 
    git clone https://github.com/SergeyKovalchuk/code-battle.git
    ```
* **Enter the folder**
     ```bash
     cd code-battle
     ```
* **Install dependencies**
     ```bash
     yarn install or npm install
     ```
* **Run project**
    ```bash
    yarn start or npm start
    ```


## To build: 

```bash
yarn  run build or npm  run build
```
